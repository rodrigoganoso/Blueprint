The reason you converted the dataframe to a wide format is because this makes it much easier to plot two columns against each other. To compare the sensor readings between DEQ and PMS5003 devices, create a scatterplot of one reading versus the other. Add a line of best fit with geom_smooth().

ggplot(pmCompareCols, aes(x = pms5003, y = deq)) + 
  geom_point() + 
  geom_smooth(method = 'lm', formula = y ~ x) + 
  labs(x = "PMS5003", y = "DEQ")

To determine the coefficients and thus the equation of the trend line, use lm(y ~ x). The term y ~ x means you are looking for an equation of best fit where y is proportional to x. This means an equation of the form y = mx + b

x <- pmCompareCols$pms5003
y <- pmCompareCols$deq

slopeIntercept <- lm(y ~ x)
print(slopeIntercept)
## 
## Call:
## lm(formula = y ~ x)
## 
## Coefficients:
## (Intercept)            x  
##     -0.6310       0.4948

The correlation coefficient tells you how strong the relationship or correlation is between x and y (the PMS5003 and the DEQ reference monitor.

print(cor(x, y))
## [1] 0.9974499

The slope and intercept are stored in the coefficients variable. Store these numbers with the names m and b. Our equation tells us what we need to do to the PMS5003 data to get the DEQ data. Now that we have the coefficients of our equation, we can transform the PMS5003 data by plugging it into the equation using mutate().

m <- slopeIntercept$coefficients[2]
b <- slopeIntercept$coefficients[1]

pmCorrected <- pmCompareCols %>%
    mutate(pmsCorrected = m * pms5003 + b)

Now plot the corrected PMS5003 values against the DEQ values.

ggplot(pmCorrected, aes(x = pmsCorrected, y = deq)) + 
    geom_point() + 
    geom_smooth(method = 'lm', formula = y ~ x) + 
    labs(x = "PMS5003 (corrected)", y = "DEQ") + 
    coord_fixed(1) + 
    theme(
        panel.grid.minor = element_blank()
    )

Finally, convert the corrected data set back into a long format. This format is useful for plotting the sensor readings side by side over time.

pmCorrectedLong <- pmCorrected %>%
    pivot_longer(cols = c(deq, pms5003, pmsCorrected),
    names_to = "sensor", values_to = "pm25")

Lastly, plot the data as a time series again. The data points will be colored based on the type of sensor making the readings. The corrected PMS5003 and DEQ data should be very similar over time.

ggplot(pmCorrectedLong, aes(x = date, y = pm25, color = sensor)) + 
    geom_point() + 
    labs(x = "Date (2020)", y = "PM2.5 Concentration", color = "Sensor type")
